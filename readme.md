## Ionic2 App

#Starfinder


### Installation

1.    ``` $ npm install npm@latest -g ```

2.    ``` $ npm install -g cordova ionic ```
    
3.    ``` $ git clone https://github.com/ElisFilipsson/Ionic2App.git ```

4.    ``` $ cd Ionic2App ```

5.    ``` $ npm install ```

6.    ``` $ ionic start ```

The page will open in your browser - Be patient.

(All pages, code etc is in /src/ folder.)
